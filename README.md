# Cypress UI Test Automation Framework

This is a demo cypress test automation framework for the following test site

https://student-portal.swaggear.life/login

Please follow the following instruction for easy set-up

## Pre-requsites

Make sure that your system have following browser isnatlled

- Google Chrome
- Mozilla Firefox ( optional )

### Installing Dependencies

Open the terminal on your VS Code and run the following command:

```shell
npm install
```

This should install all of the libarary and package dependencies for this demo project.

## Running the test

The test case be executed in few different variend.
If you want to run test by manual trigger:

```shell
npm run cy:open
```

If you want to run test on headed ( UI ) mode:

```shell
npm run cy:run
```

If you want to run test on headed (UI ) mode with Chrome

```shell
npm run cy:run:chrome
```

If you want to run test on headed (UI ) mode with Firefox

```shell
npm run cy:run:firefox
```
